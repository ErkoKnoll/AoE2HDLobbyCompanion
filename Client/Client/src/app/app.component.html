AoE 2 HD Lobby Companion uses NetHook2 to inject a DLL into Steam's address space in order to read its network traffic. While there aren't any known cases when Valve has banned for such activity then it is not warranted that they will not start taking preventative actions against users who are doing so by banning their accounts.
Most likely your account should remain intact since AoE 2 HD is not a VAC protected game nor is the software considered malicious. However it is highly recommended to close this software while playing other Steam games.
Some parts of the software such as Backend.exe and NetHook2.dll could be flagged as viruses by Anti-Virus software due to the nature of the work they have to perform, however they are merely false positives. More about NetHook2.dll in later step but Backend.exe is just a .NET web server that is spawned by the main client to perform more powerful tasks that the main client cannot do itself such as projecting an overlay.

Insert Steam API Key

Steam API key is needed to fetch additional data such as player location, total games, win ratio and drop ratio.
To get the key click here which opens a registration page. Into Domain Name insert some non existing domain (example: www.nonexistingsite.net), then check Terms of Service and then click on Register which will return a key that you can insert below.
You can skip this step and insert the key later by going into settings.
If you leave it unset or insert an incorrect key then additional data mentioned above won't be shown.
